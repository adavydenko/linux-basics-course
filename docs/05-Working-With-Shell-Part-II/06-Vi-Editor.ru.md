# Редактор Vi

- Перейти к [видеоуроку](https://kodekloud.com/topic/vi-editor/)

В этом разделе мы рассмотрим консольные текстовые редакторы Linux. `cat` удобен для короткого вывода, но неудобен для больших файлов и редактирования кода. Для этого нужны текстовые редакторы.

## Цели раздела

После раздела студент должен уметь:

- открыть файл в `vi`;
- различать командный режим, режим вставки и режим последней строки;
- вставлять и удалять текст;
- сохранять файл и выходить;
- отменять изменения;
- понимать связь `vi` и `vim`.

## Текстовый редактор

В Linux есть разные редакторы, но в этом разделе фокус на `vi`.

`vi` доступен почти во всех дистрибутивах Linux из коробки. Чтобы открыть или создать файл, выполните:

```shell
$ vi /home/michael/sample.txt
```

## Режимы Vi

У `vi` есть три основных режима.

1. Командный режим.
   - При открытии файла `vi` сначала попадает в командный режим.
   - В этом режиме клавиши воспринимаются как команды редактора.
2. Режим вставки.
   - Чтобы перейти из командного режима в режим вставки, нажмите `i`.
   - В этом режиме можно вводить текст.
   - Чтобы вернуться в командный режим, нажмите `Esc`.
   - Также можно входить в режим вставки через `I`, `o`, `O`, `a`, `A`.
3. Режим последней строки.
   - Нажмите `:`, чтобы перейти в режим последней строки.
   - Здесь можно сохранить изменения, выйти из редактора или выполнить команды редактора.
   - Чтобы вернуться в командный режим, нажмите `Esc`.

## Vim

`vim` — улучшенная версия `vi` с дополнительными возможностями, но похожим интерфейсом и режимами работы.

Во многих современных дистрибутивах команда `vi` фактически является ссылкой на `vim`.

![VIM](../../images/VIM.PNG)

## Практическое задание

1. Откройте новый файл через `vi`.
2. Перейдите в режим вставки и добавьте несколько строк.
3. Вернитесь в командный режим.
4. Удалите один символ через `x`.
5. Удалите строку через `dd`.
6. Отмените действие через `u`.
7. Сохраните файл и выйдите через `:wq`.
