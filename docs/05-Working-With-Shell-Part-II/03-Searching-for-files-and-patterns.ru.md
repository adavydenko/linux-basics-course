# Поиск файлов и шаблонов

- Перейти к [видеоуроку](https://kodekloud.com/topic/searching-for-files-and-patterns/)

В этом разделе мы рассмотрим, как искать файлы, каталоги и строки внутри файлов.

## Цели раздела

После раздела студент должен уметь:

- искать файл через `locate`;
- понимать ограничение базы `locate`;
- искать файлы через `find`;
- искать строки внутри файлов через `grep`;
- использовать основные флаги `grep`: `-i`, `-r`, `-v`, `-w`, `-A`, `-B`.

## `locate`

Если нужно найти файлы с именем `City.txt`, самый простой способ — команда `locate`:

```shell
$ locate City.txt
```

`locate` возвращает пути, совпадающие с шаблоном. Но у команды есть важное ограничение: она ищет не по файловой системе напрямую, а по базе `mlocate.db`.

Если Linux установлен недавно или файл создан только что, `locate` может ничего не найти, потому что база ещё не обновилась.

Обновить базу вручную:

```shell
$ sudo updatedb
```

Команда `updatedb` требует прав `root`.

## `find`

`find` ищет прямо в файловой системе. Укажите каталог, внутри которого нужно искать, затем условие поиска.

Поиск файла по имени:

```shell
$ find /home/michael -name City.txt
```

![locate-find](../../images/locate-find.PNG)

## `grep`

`grep` ищет строки внутри файлов. По умолчанию поиск чувствителен к регистру.

Найти слово `second` в `sample.txt`:

```shell
$ grep second sample.txt
```

Поиск без учёта регистра:

```shell
$ grep -i capital sample.txt
```

Рекурсивный поиск по каталогу:

```shell
$ grep -r "third Line" /home/michael
```

Вывести строки, которые не совпадают с шаблоном:

```shell
$ grep -v "printed" sample.txt
```

![grep](../../images/grep.PNG)

## Поиск целого слова

Чтобы найти целое слово `exam`, используйте `-w`:

```shell
$ grep -w exam examples.txt
```

Опции можно комбинировать. Например, вывести все строки, где нет целого слова `exam`:

```shell
$ grep -vw exam examples.txt
```

Показать строки после совпадения:

```shell
$ grep -A1 Arsenal premier-league-table.txt
```

Показать строки до совпадения:

```shell
$ grep -B1 4 premier-league-table.txt
```

![grep1](../../images/grep1.PNG)

`-A` и `-B` можно использовать вместе:

```shell
$ grep -A1 -B1 Chelsea premier-league-table.txt
```

![grep2](../../images/grep2.PNG)

1. Найдите файл по имени через `find`.
2. Обновите базу `locate` и найдите тот же файл через `locate`.
3. Найдите строки с `error` без учёта регистра.
4. Найдите строки, где нет слова `debug`.
5. Найдите совпадение и выведите по одной строке до и после него.
