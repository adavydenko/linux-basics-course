# Архивация и сжатие файлов

- Перейти к [видеоуроку](https://kodekloud.com/topic/file-compression-and-archival/)

В этом разделе мы разберём, как смотреть размер файлов, объединять файлы в архив и сжимать данные.

## Цели раздела

После раздела студент должен уметь:

- смотреть размер файла или каталога;
- создавать архив `tar`;
- просматривать содержимое архива;
- распаковывать архив;
- сжимать и распаковывать файлы через `gzip`, `bzip2`, `xz`;
- читать сжатые текстовые файлы без предварительной распаковки.

## Просмотр размера файлов

Команда `du` означает `disk usage` и показывает, сколько места занимает файл или каталог.

Размер в килобайтах:

```shell
$ du -sk test.img
```

Размер в удобном для чтения виде:

```shell
$ du -sh test.img
```

Размер файла также можно посмотреть через длинный список `ls -lh`:

```shell
$ ls -lh test.img
```

## Архивация файлов

`tar` объединяет несколько файлов и каталогов в один файл. Поэтому он особенно полезен для архивирования данных. Название `tar` исторически означает `tape archive`, а файлы, созданные через `tar`, часто называют `tarballs`.

Создать архив:

```shell
$ tar -cf test.tar file1 file2 file3
$ ls -ltr test.tar
```

Опции:

- `-c` — создать архив;
- `-f` — указать имя архивного файла.

Посмотреть содержимое архива:

```shell
$ tar -tf test.tar
```

Распаковать архив:

```shell
$ tar -xf test.tar
```

Создать сжатый gzip-архив:

```shell
$ tar -zcf test.tar.gz file1 file2 file3
```

## Сжатие

Сжатие уменьшает объём файла или набора данных. В Linux часто используются:

- `bzip2` — расширение `.bz2`;
- `gzip` — расширение `.gz`;
- `xz` — расширение `.xz`.

```shell
$ bzip2 test.img
$ gzip test1.img
$ xz test2.img
```

Размер получившихся файлов зависит от типа данных, алгоритма сжатия и уровня сжатия.

Распаковать такие файлы можно командами:

```shell
$ bunzip2 test.img.bz2
$ gunzip test1.img.gz
$ unxz test2.img.xz
```

## Чтение сжатых файлов

Сжатый текстовый файл не всегда нужно распаковывать на диск. Команды `zcat`, `bzcat` и `xzcat` позволяют прочитать содержимое напрямую:

```shell
$ bzcat hostfile.txt.bz2
$ zcat hostfile.txt.gz
$ xzcat hostfile.txt.xz
```

## Практическое задание

1. Создайте несколько текстовых файлов.
2. Соберите их в архив `test.tar`.
3. Посмотрите содержимое архива через `tar -tf`.
4. Распакуйте архив в отдельный каталог.
5. Создайте архив `test.tar.gz` через `tar -zcf`.
6. Сравните размеры через `ls -lh` и `du -sh`.
