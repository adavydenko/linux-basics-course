# Лабораторная работа: Vi

- Перейти к [лабораторной работе](https://kodekloud.com/topic/lab-vi-editor-3/)

## Цели лабораторной работы

После выполнения работы студент должен уметь:

- входить в режим вставки;
- возвращаться в командный режим;
- удалять символы и строки;
- сохранять файл;
- искать текст внутри файла;
- отменять изменения.

## Задания

1. Перейти в режим вставки:

   ```text
   Нажмите i
   ```

2. Выйти из режима вставки и вернуться в командный режим:

   ```text
   Нажмите Esc
   ```

3. Удалить символ:

   ```text
   Переместите курсор на нужный символ и нажмите x.
   ```

4. Изменить содержимое файла на `Welcome to KodeKloud` и сохранить файл:

   ```text
   Нажмите i, удалите старое содержимое и введите новый текст.
   Затем нажмите Esc и выполните :w!
   ```

5. В конфигурации Apache изменить порт с `80` на `5000`:

   ```text
   Перейдите в режим вставки и замените 80 на 5000 в нужной строке.
   ```

6. Удалить строку, которая начинается с `LogFormat`:

   ```text
   Перейдите на нужную строку и нажмите dd.
   ```

7. Отменить предыдущее изменение:

   ```text
   Нажмите u.
   ```

8. Найти символ `6` в файле:

   ```text
   В командном режиме выполните /6
   ```

## Что нужно сдать

- Краткий список использованных команд `vi`.
- Итоговый файл после редактирования.
- Ответ, какой командой выполняется поиск в `vi`.
