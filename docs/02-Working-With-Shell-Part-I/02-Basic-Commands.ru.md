# Базовые команды Linux

- Перейти к [видеоуроку](https://kodekloud.com/topic/basic-linux-commands-3/)

## Цели раздела

После раздела студент должен уметь:

- определять текущий рабочий каталог;
- просматривать содержимое каталогов;
- создавать файлы и каталоги;
- перемещаться по абсолютным и относительным путям;
- копировать, перемещать, переименовывать и удалять файлы;
- выбирать подходящую команду для просмотра содержимого файла.

В этом разделе мы рассмотрим базовые команды Linux,
- в частности, навигацию по файловой системе и создание новых файлов и каталогов.
- Мы сделаем это, выполнив небольшое практическое задание в оболочке Linux.

#### Наша цель — создать следующую структуру каталогов. Верхний каталог — это домашний каталог пользователя **`/home/michael`** (он уже существует), всё остальное ниже нужно создать.

![mkdir_cd_working_with_shell_I](../../images/mkdir_cd_working_with_shell_I.PNG)

Чтобы вывести текущий рабочий каталог, выполните команду **`pwd`**
```shell
$ pwd
```

Чтобы увидеть содержимое каталога, выполните команду **`ls`**
```shell
$ ls 
```

Чтобы создать каталог, выполните команду **`mkdir`**
```shell
$ mkdir Asia
```

Чтобы создать несколько каталогов, выполните **`mkdir`** с перечислением имён: **`<dir1> <dir2> .. <dirN>`**
```shell
$ mkdir Europe Africa America
```

Чтобы перейти в подкаталог из текущего каталога, выполните **`cd <directory_name>`**
```shell
$ cd Asia
```

Чтобы рекурсивно создать вложенные каталоги, выполните **`mkdir -p <parent>/<child>`**
```shell
$ mkdir -p India/Mumbai
```

Чтобы подняться на уровень выше, выполните **`cd ..`**
```shell
$ cd ..
```

Чтобы вернуться в домашний каталог текущего пользователя из любого места, выполните **`cd`**
```shell
$ cd
```

#### Абсолютные и относительные пути

![Absolute_and_relative_path_working_with_shell_I](../../images/Absolute_and_relative_path_working_with_shell_I.PNG)

**Разница между абсолютным и относительным путями**

- **Абсолютный путь**: путь от корня файловой системы `/` до файла или каталога.
- **Относительный путь**: путь относительно текущего рабочего каталога (`pwd`).

Перейти в каталог по абсолютному пути:
```shell
$ cd /home/michael
```

Перейти в каталог по относительному пути:
```shell
$ cd Asia
```

#### Альтернативы команде **`cd`**

![pushd_popd](../../images/pushd_popd.PNG)

Альтернатива **`cd`** — команды **`pushd`** и **`popd`**. Чтобы перейти в каталог, используйте **`pushd <directory_name>`**
```shell
$ pushd /etc
```

Можно переходить дальше в подкаталоги, накапливая стек директорий:
```shell
$ pushd /var
$ pushd /tmp
$ pwd
/tmp
```

Чтобы вернуться назад по стеку (например, в исходный каталог), используйте **`popd`**
```shell
$ popd
```

#### Ещё базовые команды Linux

Для примеров используем ту же структуру, но с добавленными файлами и каталогами, как на схеме. Цель — привести структуру к виду на «после» диаграмме.

![before_after_commands](../../images/before_after_commands.PNG)

Переместить файл или каталог — **`mv <source> <destination>`**
```shell
$ mv /home/michael/Europe/Morocco /home/michael/Africa/   # Абсолютный путь
$ mv Europe/Morocco Africa/                               # Относительный путь
```

Переименовать каталог — **`mv <oldname> <newname>`**
```shell
$ mv Asia/India/Munbai Asia/India/Mumbai
```

Скопировать файл в каталог — **`cp <filename> <destination_dir>`**
```shell
$ cp Asia/India/Mumbai/City.txt Africa/Egypt/Cairo
```

Удалить файл — **`rm /path/<filename>`**
```shell
$ rm Europe/UK/London/Tottenham.txt
```

Рекурсивно скопировать каталог — **`cp -r <source> <destination>`**
```shell
$ cp -r Europe/UK Europe/UnitedKingdom
```

Вывести содержимое файла — **`cat /path/to/<filename>`**
```shell
$ cat Asia/India/Mumbai/City.txt
```

Записать содержимое в файл через `cat` (перенаправление) — **`cat > /path/to/<filename>`**
```shell
$ cat > Africa/Egypt/Cairo/City.txt
  Cairo
  # Нажмите Ctrl + d для завершения ввода
```

Создать пустой файл — **`touch /path/to/filename`**
```shell
$ touch /home/michael/Asia/China/Country.txt
```

Постраничный просмотр файла — **`more /path/to/filename`** (не рекомендуется для больших файлов)
```shell
$ more new_file.txt
```

Продвинутый постраничный просмотр и навигация — **`less /path/to/filename`**
```shell
$ less new_file.txt
```

Длинный список файлов и каталогов — **`ls -l`**
```shell
$ ls -l
```

Показать все файлы, включая скрытые — **`ls -la`**
```shell
$ ls -a
```

Список по времени изменения (новые первыми) — **`ls -lt`**
```shell
$ ls -lt
```

Список от старых к новым — **`ls -ltr`**
```shell
$ ls -ltr
```

## Практическое задание

1. Создайте в домашнем каталоге структуру `course/linux/lab01`.
2. Перейдите в каталог `lab01` сначала по абсолютному пути, затем по относительному.
3. Создайте файл `notes.txt` через `touch`.
4. Запишите в файл одну строку через `cat > notes.txt`.
5. Скопируйте файл в `notes-copy.txt`.
6. Переименуйте `notes-copy.txt` в `notes-backup.txt`.
7. Просмотрите каталог через `ls -la`, `ls -lt` и `ls -ltr`.
8. Удалите резервную копию и проверьте результат.
