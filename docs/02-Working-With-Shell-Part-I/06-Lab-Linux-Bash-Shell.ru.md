# Лабораторная работа: Bash

- [Доп. лабораторная](https://kodekloud.com/topic/lab-linux-bash-prompt/)

## Цели лабораторной работы

После выполнения работы студент должен уметь:

- проверять текущую оболочку пользователя;
- менять оболочку пользователя;
- читать и создавать переменные окружения;
- проверять переменную `PATH`;
- создавать постоянные alias;
- настраивать приглашение Bash.

## Задания

В этой работе вы пройдёте весь типичный цикл настройки рабочего окружения: узнаете и смените оболочку, прочитаете и создадите переменные, разберётесь с `PATH` и настроите приглашение. Главная сквозная идея — отличать временные изменения от постоянных: то, что нужно сохранить между сессиями, попадает в `~/.profile`.

1. Проверить оболочку, назначенную текущему пользователю. Важно понимать тонкость: эта команда показывает оболочку, *заданную* пользователю в системе, но не обязательно ту, что прямо сейчас выполняет ваши команды:

   ```shell
   $ echo $SHELL
   ```

2. Изменить оболочку пользователя `bob` с Bash на Bourne Shell. Флаг `-s` сразу задаёт новую оболочку, не переходя в диалоговый режим:

   ```shell
   $ chsh -s /bin/sh bob
   ```

3. Узнать значение переменной окружения `TERM` — в ней хранится тип терминала, с которым вы работаете:

   ```shell
   $ echo $TERM
   ```

4. Создать переменную окружения `PROJECT=MERCURY` и сразу сделать её постоянной, дописав в `~/.profile`. Двойная стрелка `>>` добавляет строку в конец файла, не затирая его содержимое:

   ```shell
   $ echo export PROJECT=MERCURY >> ~/.profile
   ```

5. Просмотреть `PATH` и определить, какого из каталогов в нём нет. Ответ для этой работы:

   ```text
   /opt/caleston-code
   ```

6. Создать alias `up` для команды `uptime` и тоже закрепить его в `~/.profile`, чтобы он был доступен в новых сессиях:

   ```shell
   $ echo alias up=uptime >> ~/.profile
   ```

7. Изменить приглашение пользователя `bob`, чтобы оно показывало дату в таком формате:

   ```text
   [Wed Apr 22]bob@caleston-lp10:~$
   ```

   Изменение должно быть постоянным.

   ```shell
   $ PS1='[\d]\u@\h:\w\$'
   ```

   Или добавить настройку в `~/.profile`, чтобы она сохранялась между сессиями:

   ```shell
   $ echo "PS1='[\d]\u@\h:\w\$'" >> ~/.profile
   ```

## Что нужно сдать

Оформите отчёт по [единому формату](../00-Lab-Environment/01-Lab-Environment-Setup.ru.md#формат-отчёта-по-лабораторной-работе). Дополнительно для этой работы приложите:

- Вывод `echo $SHELL`, `echo $TERM`, `echo $PATH`.
- Команды, которыми были добавлены `PROJECT`, alias `up` и `PS1`.
- Краткое объяснение, почему изменения в `~/.profile` сохраняются между сессиями.
