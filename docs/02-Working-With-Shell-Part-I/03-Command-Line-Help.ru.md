# Справка в командной строке

- Перейти к [видеоуроку](https://kodekloud.com/topic/command-line-help-3/)

В этом разделе мы разберём, как получать справку по командам прямо из терминала.

## Цели раздела

После раздела студент должен уметь:

- быстро узнавать назначение команды;
- открывать подробную справочную страницу;
- искать команды по ключевому слову;
- отличать краткую справку от полной документации.

## Зачем нужна справка

Невозможно запомнить все команды Linux и все их опции. Нормальная рабочая привычка — уметь быстро найти справку, проверить синтаксис и понять, какие флаги нужны для конкретной задачи.

## `whatis`

Команда `whatis` показывает краткое описание команды в одну строку.

Синтаксис:

```shell
$ whatis <command>
```

Пример:

```shell
$ whatis date
```

Эта команда удобна, когда нужно быстро вспомнить назначение утилиты.

## `man`

Большинство внутренних и внешних команд поставляются со справочными страницами `man pages`. Они содержат подробное описание команды, синтаксис, опции и часто примеры использования.

Синтаксис:

```shell
$ man <command>
```

Пример:

```shell
$ man date
```

Внутри `man` обычно можно использовать:

- `Space` для перехода на страницу ниже;
- `b` для перехода на страницу выше;
- `/text` для поиска по странице;
- `q` для выхода.

## `-h` и `--help`

Многие команды поддерживают краткую справку через опции `-h` или `--help`.

```shell
$ date -h
$ date --help
```

Такой вариант часто быстрее, чем `man`, если нужно посмотреть только доступные параметры.

## `apropos`

Команда `apropos` ищет по названиям и описаниям справочных страниц. Она полезна, если вы не помните точное имя команды, но знаете ключевое слово.

Синтаксис:

```shell
$ apropos <keyword>
```

Пример:

```shell
$ apropos modpr
```

Если поиск ничего не нашёл, проверьте написание ключевого слова или попробуйте более общее слово.

## Практическое задание

1. Узнайте назначение команд `date`, `passwd`, `uptime` через `whatis`.
2. Откройте `man date` и найдите внутри страницы слово `format`.
3. Сравните вывод `date --help` и `man date`.
4. Найдите команды, связанные с password, через `apropos password`.
