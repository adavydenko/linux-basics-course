# Современная сеть, контейнеры, диагностика

<div class="part-intro">

Современный сетевой стек Linux (`ip`, `ss`), постоянная конфигурация через Netplan и NetworkManager, контейнеры (Docker, Dockerfile), общий подход к диагностике системы. Часть завершается итоговым проектом, в котором собираются все навыки курса: студент разворачивает рабочее окружение на учебном сервере, поднимает службу, настраивает её автозапуск и диагностику.
