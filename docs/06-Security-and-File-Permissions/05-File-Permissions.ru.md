# Права доступа к файлам

- Перейти к [видеоуроку](https://kodekloud.com/topic/file-permissions-and-ownership/)

В этом разделе мы рассмотрим идентификаторы типов файлов и права доступа Linux для файлов и каталогов.

## Цели раздела

После раздела студент должен уметь:

- читать права доступа в выводе `ls -l`;
- понимать права `read`, `write`, `execute`;
- изменять права через `chmod`;
- менять владельца и группу через `chown` и `chgrp`;
- понимать числовую форму прав.

## Права каталогов

Посмотреть права каталога:

```shell
$ ls -ld /home/bob/random_dir
```

Узнать текущего пользователя:

```shell
$ whoami
```

Перейти в каталог:

```shell
$ cd /home/bob/random_dir
```

Для каталогов `x` означает возможность войти в каталог и проходить через него по пути.

## Права файлов

Права Linux обычно читаются как три группы: владелец, группа, остальные.

![filep](../../images//filep.PNG)

## Изменение прав: `chmod`

Дать владельцу полный доступ:

```shell
$ chmod u+rwx test-file
```

Дать чтение владельцу, группе и остальным, убрать выполнение:

```shell
$ chmod ugo+r-x test-file
```

Убрать все права у остальных:

```shell
$ chmod o-rwx test-file
```

Владелец получает полный доступ, группа — чтение без выполнения, остальные — без доступа:

```shell
$ chmod u+rwx,g+r-x,o-rwx test-file
```

Полный доступ владельцу, группе и остальным:

```shell
$ chmod 777 test-file
```

Чтение и выполнение владельцу, группе и остальным:

```shell
$ chmod 555 test-file
```

Чтение и запись владельцу и группе, без доступа для остальных:

```shell
$ chmod 660 test-file
```

Полный доступ владельцу, чтение и выполнение группе, без доступа для остальных:

```shell
$ chmod 750 test-file
```

## Изменение владельца

Изменить владельца на `bob`, а группу на `developer`:

```shell
$ chown bob:developer test-file
```

Изменить только владельца файла на `bob`, группу не менять:

```shell
$ chown bob android.apk
```

Изменить группу файла `test-file` на `android`:

```shell
$ chgrp android test-file
```

## Практическое задание

1. Создайте файл и посмотрите его права через `ls -l`.
2. Дайте владельцу выполнение через `chmod u+x`.
3. Уберите все права у остальных через `chmod o-rwx`.
4. Установите права `750` и объясните каждую цифру.
5. Измените группу файла через `chgrp`.
