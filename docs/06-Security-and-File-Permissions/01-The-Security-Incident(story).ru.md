# История: инцидент безопасности

- Перейти к [истории](https://kodekloud.com/topic/security-incident/)

Эта история вводит раздел о пользователях, группах, правах доступа, SSH и базовой сетевой фильтрации. В Linux безопасность начинается с простых вещей: кто вошёл в систему, какие у него права и какие файлы он может читать или изменять.
