# Приложение. Шпаргалка по командам Linux

Это краткий справочник по основным командам, которые встречаются в пособии. Он не заменяет разделы курса, а помогает быстро вспомнить нужную команду и её назначение. Подробности по каждой команде — в соответствующем разделе и во встроенной справке (`man <команда>` или `<команда> --help`).

Команды сгруппированы по темам в том же порядке, что и модули курса. Угловые скобки `<...>` обозначают то, что нужно подставить самому.

## Навигация и каталоги

| Команда | Что делает |
| --- | --- |
| `pwd` | показать текущий рабочий каталог |
| `ls` | показать содержимое каталога |
| `ls -l` | подробный список (права, владелец, размер, дата) |
| `ls -la` | то же, включая скрытые файлы |
| `ls -ltr` | подробный список по времени, от старых к новым |
| `cd <каталог>` | перейти в каталог |
| `cd ..` | подняться на уровень выше |
| `cd` | вернуться в домашний каталог |
| `pushd <каталог>` | перейти в каталог и запомнить текущий в стеке |
| `popd` | вернуться к предыдущему каталогу из стека |
| `mkdir <имя>` | создать каталог |
| `mkdir -p <путь>` | создать цепочку вложенных каталогов |
| `tree` | показать дерево каталогов |

## Файлы

| Команда | Что делает |
| --- | --- |
| `touch <файл>` | создать пустой файл или обновить дату изменения |
| `cp <источник> <назначение>` | скопировать файл |
| `cp -r <источник> <назначение>` | скопировать каталог рекурсивно |
| `mv <источник> <назначение>` | переместить или переименовать |
| `rm <файл>` | удалить файл (необратимо) |
| `rm -r <каталог>` | удалить каталог со всем содержимым |
| `cat <файл>` | вывести содержимое файла целиком |
| `more <файл>` | постраничный просмотр |
| `less <файл>` | постраничный просмотр с прокруткой и поиском |
| `head -n <N> <файл>` | первые N строк файла |
| `tail -n <N> <файл>` | последние N строк файла |
| `file <файл>` | определить тип файла |
| `ln -s <цель> <ссылка>` | создать символическую ссылку |

## Поиск файлов и текста

| Команда | Что делает |
| --- | --- |
| `find <путь> -name <имя>` | найти файлы по имени |
| `locate <имя>` | быстрый поиск по индексу имён файлов |

## Перенаправление и конвейеры

| Конструкция | Что делает |
| --- | --- |
| `>` | перенаправить вывод в файл (перезаписать) |
| `>>` | перенаправить вывод в файл (дописать) |
| `2>` | перенаправить поток ошибок (STDERR) |
| `<` | взять ввод из файла |
| `\|` | передать вывод одной команды на вход другой |
| `&&` | выполнить следующую команду, если предыдущая успешна |
| `\|\|` | выполнить следующую команду, если предыдущая завершилась с ошибкой |

## Архивация и сжатие

| Команда | Что делает |
| --- | --- |
| `tar -cf <архив> <путь>` | собрать архив |
| `tar -xf <архив>` | распаковать архив |

| `zcat <файл.gz>` | прочитать сжатый файл без распаковки |
