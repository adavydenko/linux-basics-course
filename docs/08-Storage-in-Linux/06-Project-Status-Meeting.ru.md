# Статусная встреча проекта

- Перейти к [встрече](https://kodekloud.com/topic/project-status-meeting-story/)

Команда собирается на статусную встречу — подвести итоги по работе с хранилищем. Здесь всё, что вы прошли по отдельности, складывается в одну картину: диск размечают на разделы, на разделе создают файловую систему, каталоги отдают по сети через NFS, а поверх дисков выстраивают гибкий слой LVM.

Эта короткая история завершает раздел о хранилищах. Если на встрече каждый из этих шагов кажется понятным и вы видите, как они соединяются друг с другом, — материал усвоен.
