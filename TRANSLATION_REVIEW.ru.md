## Оставшиеся методические риски

- Разделы по `iptables` и классическим сетевым командам важны для понимания старых систем, но студентам нужно явно объяснять связь с современными `nftables`, `ufw`, `firewalld`, `ip` и `ss`.
- Темы хранения данных и LVM требуют изолированной лабораторной среды, иначе студенты могут повредить рабочую систему.
- Контейнеры и Git из командной строки добавлены как прикладные темы; их глубину стоит согласовать с общей программой факультета.
- Python из командной строки и `venv` оставлены как практический минимум для вычислительных задач, а не как отдельный курс программирования.
